# 📅 Playzio BARA

Application web pour organiser des parties de **Tennis**, **Padel** et **Soirées** entre amis.

## 🚀 Démarrage rapide

```bash
# Installation
npm run install:all

# Développement
npm run dev
```

- **Frontend** : http://localhost:5173
- **Backend** : http://localhost:8080

## ✨ Fonctionnalités

- 🔐 Connexion sécurisée
- 📅 Calendrier interactif
- 👥 Gestion des amis et groupes
- 🎾 Création de disponibilités
- 📱 Interface responsive

## 🎨 Design

Thème sombre avec gradients beige-violet (#d4af8c → #8a2be2)

## 🔧 Technologies

- **Frontend** : React + Vite
- **Backend** : Node.js + Express
- **Base de données** : JSON
- **Déploiement** : Vercel + Railway
